# Fix AI-Written Code

> Copy-paste prompts for fixing, securing, testing, and productionizing code written by AI coding agents.

AI coding agents can generate code quickly, but they can also leave behind bugs, weak security, fake tests, inconsistent architecture, and production risks.

This repo gives you practical prompts you can paste into Codex, Claude Code, Cursor, Copilot, or other coding agents to clean up and harden a codebase.
